# 介绍

vue 项目模板，用于测试翻译效果

## 安装

装项目依赖

```
npm i
```

装命令行工具

```
npm i -g @ovointl/i18n-extract-cli
```

## 运行项目

在当前目录(vue-demo)下执行

```
npm start
```
